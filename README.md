# React Boilerplate

A custom made boilerplate, suited for building SPAs from scratch.\
I built this as an alternative to create-react-app.

# Core Features

- React
- TypeScript
- SASS/ SCSS Loader
- CSS Loader
- Dark Mode + Light Mode (Theme)
- Image Loader
- Font Loader
- ESLint
- Webpack Bundling
- LESS Loader (optional)

# Installation

- Clone the repo.
- Install dependencies with yarn or npm.
